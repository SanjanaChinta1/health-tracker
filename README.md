# health-tracker


**Rural Health Tracker & Awareness**

**About the Project**

The Rural Health Tracker & Awareness App is a simple application designed to support basic healthcare awareness in rural areas. It helps users maintain health records, receive reminders, and learn important health information in an easy and accessible way.



**Features**

-Basic health record management
-Vaccination and check-up reminders
-Support for local languages
-Simple and user-friendly interface
-Health awareness content

**Purpose**

This project aims to improve health awareness among rural communities and assist health workers in maintaining basic digital health records.


**Technology Used**

Mobile application framework
Local database for offline storage
Simple backend services


**Future Enhancements**

-Add more regional languages
-Improve user interface
-Add more health awareness content
-Better reminder management


**Usage**

This application can be used by rural families, health workers, and community centers to support basic healthcare tracking and awareness.


**To run**

Install all the requirements

run this in terminal:

streamlit run app.py
